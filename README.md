## Installation:
**Open terminal and type:**
```shell script
  git clone https://github.com/BLINKING-IDIOT/Aliens_eye
  cd Aliens_eye
  bash install.sh
```
* Now Aliens_eye is Succesfully installed in your system

* Enter the command "aliens_eye" to run the tool(without "")
